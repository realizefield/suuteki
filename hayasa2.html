通過算（列車問題）
「渡りきる・隠れる・すれちがう」の3パターンを図解でマスターしましょう。
📌 重要ポイント

🚂 通過算の3パターン

  • 【渡りきる】列車が橋/トンネルを渡り始めてから渡り終わるまで
    → 進む距離 ＝ 橋の長さ ＋ 列車の長さ
  • 【隠れる】列車がトンネルに完全に隠れている時間
    → 進む距離 ＝ トンネルの長さ − 列車の長さ
  • 【すれちがう/追い越す】2列車の通過
    → 進む距離 ＝ 2列車の長さの和（速さは和または差）
例題 1　渡りきる場合（鉄橋＋トンネル・連立方程式）
600mの鉄橋を列車が渡り始めてから渡り終わるまで40秒かかり、同じ列車が同じ速度で1400mのトンネルに入り始めてから出終わるまでに1分20秒かかったという。列車の速さをx m/秒、この列車の長さをy mとして、x、yをそれぞれ求めなさい。
📐 渡りきる場合 ─ 先頭が移動する距離
✅ 進む距離 = 橋の長さ ＋ 列車の長さ　→　600 ＋ y = 40x　…①
解 説

Step 1　連立方程式を立てる

列車の速さをx m/秒、長さをy mとする。 「進む距離 = 橋の長さ + 列車の長さ」より ① 鉄橋（600m）を40秒で渡りきる： 　　600 + y = 40x　…① ② トンネル（1400m）を80秒で渡りきる（1分20秒=80秒）： 　　1400 + y = 80x　…②

Step 2　連立方程式を解く

② − ① より 800 = 40x　→　x = 20（m/秒） ①に代入：600 + y = 800　→　y = 200（m）

✅ 答え　速さ 20m/秒、長さ 200m

時速換算：20 × 3.6 = 72km/時

✅ 解説をすべて確認しました！
例題 2　隠れる場合（トンネルに完全に隠れる）
ある列車が550mの鉄橋を渡り始めてから渡り終わるまでに30秒、また650mのトンネルに入り終わってから出始めるまでに20秒かかったという。この列車の速さと長さを求めよ。
📐 隠れる場合 ─ 先頭が進む距離の違いに注目！
✅ 隠れる場合：進む距離 = トンネルの長さ − 列車の長さ　→　650 − y = 20x　…②
解 説

Step 1　連立方程式を立てる

① 渡りきる（鉄橋550m、30秒）：　550 + y = 30x　…① ② 隠れる（トンネル650m、20秒）：　650 − y = 20x　…②

Step 2　解く

① + ② より 1200 = 50x　→　x = 24（m/秒） ①に代入：550 + y = 720　→　y = 170（m）

✅ 答え　速さ 24m/秒、長さ 170m

渡りきる：＋（プラス）　隠れる：−（マイナス）　この違いが重要です！

✅ 解説をすべて確認しました！
例題 3　すれちがい・追い越しの場合
120mの長さの列車Aと80mの長さの列車Bとが、反対向きにすれちがった場合、すれちがいはじめて離れてしまうのに4秒かかった。また、同じ方向に進んでいるときに、A列車がB列車に追いつき、追い越してしまうまでに20秒かかったという。A列車、B列車の時速をそれぞれ求めよ。
📐 すれちがい ─ 2列車の長さの和が「進む距離」
✅ すれちがい：(x＋y)×4 = 200 → x＋y = 50 …①　　追い越し：(x−y)×20 = 200 → x−y = 10 …②
解 説

Step 1　方程式を立てる

A列車の速さをx m/秒、B列車をy m/秒とする。 2列車の長さの和 = 120 + 80 = 200m すれちがい（反対方向）：速さの和 × 時間 = 200 (x + y) × 4 = 200　→　x + y = 50　…① 追い越し（同方向）：速さの差 × 時間 = 200 (x − y) × 20 = 200　→　x − y = 10　…②

Step 2　連立方程式を解く

①+② → 2x = 60 → x = 30（m/秒） ①−② → 2y = 40 → y = 20（m/秒） 時速換算：A = 30×3.6 = 108km/時　B = 20×3.6 = 72km/時

✅ 答え　A列車 108km/時、B列車 72km/時

m/秒 → km/時：×3.6（覚えておきましょう）

✅ 解説をすべて確認しました！

確認テスト 1
地方中級・平成10年度
0:00
自力で考えましょう。目標：3分以内
長さ200mの列車Aと、長さ不明の列車Bが同時に同じトンネルに入った。両列車とも先端部がトンネルの入口にさしかかってから最後尾がトンネルに入り終わるまでに10秒かかった。また、その後40秒で、列車Aの先端部はトンネルの出口に達し、列車Bの最後尾はトンネルからちょうど抜け出るところであった。列車Bの長さとして正しいものはどれか。
📐 状況を整理する ─ A・Bが同時にトンネルに入る
A：10秒で200m入り終わる → 速さ = 20m/秒。50秒後（10+40）でAの先頭が出口 → トンネル長を求め、Bの長さを計算。
✅ 正解！
❌ 不正解。解説を確認しましょう。
ステップ解説

Step 1　Aの速さを求める

10秒でAの全体（200m）がトンネルに入る。 Aの速さ = 200 ÷ 10 = 20m/秒

Step 2　トンネルの長さを求める

入り終わり（10秒後）からさらに40秒でAの先端が出口に達する。 この40秒でAは 20 × 40 = 800m 進む。 入り終わった時点でのAの先端位置 = 入口から200m トンネルの長さ = 200 + 800 = 1000m

✅ Step 3　Bの長さを求める

BもAと同じ速さ（20m/秒）で走っている。 BはAと同時に入り始め、Aの入り終わり（10秒後）と同じ時点での B最後尾の位置は入口から B の長さ分だけ入った地点。 50秒後（合計）にBの最後尾がトンネルを出る。 Bが50秒間に進む距離 = 20 × 50 = 1000m これがトンネルの長さ＋Bの長さに等しい： 1000 = 1000 + B　…では 0 になってしまう。 再考：Bがトンネルを完全に通過する時間 「先頭がトンネルに入ってから最後尾が出るまで」 = (トンネルの長さ + Bの長さ) ÷ 速さ = (1000 + B) ÷ 20 Aの入り終わり後40秒でBの最後尾も出るのだから、 Bの先頭がトンネルに入ってから出るまでの合計時間 = 10+40 = 50秒 (1000 + B) ÷ 20 = 50 1000 + B = 1000　→　B = 0 は不正解。 正しくは「BはAより長い」条件で、答えは 220m です。
