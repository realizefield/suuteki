Step 1　C を確定する ─ 1行目の一の位

1行目 ABC×2 の結果は □□A0。一の位が 0 なので，C×2 の一の位が 0 でなければなりません。 0 と 2 は使用不可なので，候補は C=5（5×2=10）のみ。 → C = 5 確定。

Step 2　1行目の十の位から A の条件を導く

C=5 なので掛けられる数は AB5。 AB5×2 の十の位を考えます。 C×2=10 なので一の位は0，十の位へ1繰り上がります。 B×2＋1（繰り上がり分）を計算し，その一の位が A になります。 → 2B＋1 の一の位 = A　…①

Step 4　①②の連立で B・A を確定

①と②はどちらも「一の位が A」なので等しくなります。 2B＋1 の一の位 = 5B＋2 の一の位 → 差をとると 3B＋1 の一の位が 0 → 3B＋1 = 10　→ 3B = 9　→ B = 3 ① に B=3 を代入：2×3＋1 = 7　→ 一の位は 7　→ A = 7 → A=7，B=3，C=5 が確定。

ABC = 735 で全行を確認： 　735 × 2 = 1470　→ □□70 ✓ 　735 × 5 = 3675　→ 3□75 ✓ 　1470 + 36750 = 38220　→ 3□220 ✓ A+B−C = 7+3−5 = 5　　正解：選択肢 1
